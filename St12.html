📜ናይ ሓሶት ዜና ማለት ንስኻ/ኺ ዘይተሰማማዕካሉ ዜና ማለት ኣይኮነን ። ብውልቂ ይኹን ብትካል ዝዝርጋሕ ዝተፈብረኸ ዝተጋነነ መደናገሪ ዝኾነ ሓበሬታ ማለት'ዩ
📃🕥 ናይ ሓሶት ዜና/ወረ ኮነ ተባሂሉ ዝተጋገየ ዝተፈብረኸ ዋላሓደ ክረጋገጽ ዘይክእል ምንጪ ዘይብሎም ሓበሬታ ንሰማዒ ወይ ኣንባቢ ንምግጋይ ተባሂሉ ዝዝርጋሕ ወረ/ ሓበሬታ እዩ ። እዚ ጉዳይ ብጽሑፍ : ብዜና :ብሕትመት ብዝርርብ : ብኤሌክትሮኒካዊ ዲጂታላዊ መራኸቢታት መልክዕ ክቀርብ ይክእል ። ንሰባት ይኹን ንሓደ ኣካል ዝንኡ ንምጉዳልን : ነቲ ሓቀኛ ዝኾነ ሓበሬታ ንምሽፋንን ብመወዓውዒ ኣታዊ ንምርካብ ተባሂሉ ክስራሕ ይኽእል ኣድልዎ ምግባር ዝተጋገየ ቀመር ብሓሶት ምስፍሕፋሕ ምስ ናይ ሓሶት ሓበሬታታት ይጽብጸብ ።
🕥 ዝተጋገየ ሓበሬታ ብምዝርጋሕ ናይ ሕዝቢ ርእይቶ ብምቕራጽ ወይ ብምልዋጥ ዝዓለመ እዩ ። ኣብ ሕክምናዊ ምርምራት : ኣብ ፖለቲካ :ኣብ ኣክስዮን ኣብ ንግዳዊ ንጥፈታትን ካልእጉዳያት ዝውቱር እዩ ።
🕥 ናይ ሓሶት ሓበሬታታት ካብቲ ሓቀኛ ሓበሬታታት ንላዕሊ ተፈታውነት ኣለዎም ። ምኽንያቱ መራኸቢ ብዙሓን እቲ ሰብ እንታይ ይደሊ መጽናዕቲ ኣካይደን ነቲ ዜናታት ስለ ዘዳልውዎ ናይ ሓሶት ዜናታት ከስፋሕፍሕ ዓቕሚ ኣለዎ
🕥ብዙሕ ግዜ ዝምስጥ ኣርእስቲ ስለ ዘለዎም ሰባት ነቲ ሓበሬታ ንምርኣዩ ይህንጠዩ ። ብዝተገብረ መጽናዕቲ ኣብ ከም ፌስቡክ ትዊተር ኣብ ዝኣመሰለ መራኸቢታት ካብቲ ሓቀኛ ንላዕሊ እቲ ናይ ሓሶት ሰለስተ ዕጽፊ ከም ዝዝርጋሕ እቲ መጽናዕቲ ይሕብር ። እዚ ከኣ ምስ ወድ ሰብ ባህርይ ዝተተሓሓዘ እዩ ። ደቂ ሰባት ኣገረምቲ : ሓደስቲ ዝኾኑ ፍጻመታት ይምስጦም እዩ ።
📃ናይ ሓሶት ዜናታት ብዙሕ እዋን ካብቲ ክውንነታዊ ወይ ሓቀኛ ንላዕሊ ስለ ዘሕጉስ ። ብከምዚ ምኽንያት ኣብ ውሽጢ ሓንጎልና ምንቕስቃስ ምንቕቓሕ ትፈጥር ። ብተወሳኪ Motivated ዝኾነ ኣተሓሳስባ እውን ነቲ ናይ ሓሶት ሓበሬታታት ኣብ ምልዕዓል ተረኺቡ እዩ ።
📜 ዓበይቲ ሓሶታት
🕕 ኣሜሪካውያን ኣብ ልዕሊ ጸለምትን ( black )ቀያሕቲ ህንዳውያን ( Native ) ኣዝዩ ብዙሕ ዝኾነ ጸለመታትን ናይ ሓሶት ሓበሬታታትን ዘርጊሓ ።
📃🕥 ኣብዚ ቀረባ ግዜ ብ2016 ኣብ ፕረዝደንታዊ ምርጫ ትራንፕ/ሂላሪ ብሚልዮናት ዝግመት ናይ ሓሶት ዝኾኑ ፌስቡክን ትዊተርን ብምኽፋት ነቲ ምርጫ ጸልይዎ እዩ ተባሂሉ ብዙሕ ኣዘራራቢ ኣጀንዳ ኮይኑ ነይሩ ።
📃🕥 ሂትለር ብ1933 ናብ ስልጣን ምስ መጸ ንጆሴፍ ጎብልስ ሚንስተር ፕሮፖጋንዳ ገይሩ ሸይምዎ ። ጆሴፍ ጎብልስ ኣብ ውሽጥን ደገን ብዙሓት ጸላእቲ ኣለውና ብምባል ናይ ሕትመት ብሮድካስት ብምውናን ወይ ፖለቲካዊ ተጽዕኖ ብምግባር ናይ ሓሶትን ፕሮፖጋንዳ ሓበሬታታት እንዳ ዘርግሑ ነቲ ህዝቢ የእሚኖሞ ነበሩ ። እንግሊዝ ብወገና ንወተሃደራት ጀርመን ተስፋ ዘቑርጽ ናይ ራድዮ ፈነወን በራሪ ወረቀትን ትዝርግሕ ነይራ ።
📜 ዓይነት ሓሶታት
🔎 1, Satire or Parody ኣስሓቒ/ መስሓቕ ጉድኣት ንኸብጽሕ ድልየት የብሉን መተዓሻሸዊ ክኸውን ግን ዓቕሚ ኣለዎ ።
🔎 2, False Connection ናይ ሓሶት ርክብ ኣርእስተ ዜናታት : ምስላዊ መግለጺታት ወይከኣ መግለጺኡ ነቲ ትሕዝቶ ምስ ዘይድግፍ ።
🔎 3, Misleading Content መጋገይ ትሕዝቶ ናይ ሓበሬታ ኣጠቃቕማ ዝተጋገየ ክኸውን እንከሎ
🔎 4, False Context ናይ ሓሶት ኣውድ
🔎 ሓቀኛ ትሕዝቶ ምስ ሓሶት ዝኾነ ሓበሬታ ክሕወስ እንከሎ 5, Impostors Content ኣምሰሉ ትሕዝቶ ሓቀኛ ምንጭታት ከምስሉ እንከለው ( ካብ ምንጩ ብሓሶት ዝተዘርግሐ )
🔎 6, Manipulated ዝንቡዕ ሓበሬታ ሓቀኛ ምስሊ ወይ ሓበሬታ ንምትላል ክስራሕ እንከሎ
🔎7, Fabricated Contored ዝተቐነባበረ ትሕዝቶ ሓድሽ ትሕዝቶ 100% ንምግጋይ ንምጉዳእ ዝተዓለመ ሓሶት እዩ ።
📜 ዜና/ ሓበሬታ ኽንረክብ እንከለና ሓቂ/ሓሶት ከም ዝኾነ ብኸምዚ ንገምግም
💡 1, ምንጪ ሓበሬታ ኣብ ግምት ምእታው ( እቲ ተልእኾ ወይ ዕላማ ንምርዳእ )
💡 2, ካብቲ ኣርእስቱ ንላዕሊ ነቲ ትሕዝቶ ምንባብ ( ኣጠቓላሊ እቲ ታሪኽ/ ሓበሬታ ንምፍላጥ )
💡3, ዝጸሓፎ ሰብ ፈትሽ ( ሓቀኛን ተኣማኒን ከም ዝኾነ ንምርግጋጽ ተመልከት )
💡4, ደገፍቲ ምንጭታት ምግምጋም ( ነቲ ሕቶታት ዝድግፍ ምኳኖም ንምርግጋጽ ) Assess the supporting sources ( to ensure they support the claims )
💡5, ዝተሓትመሉ መዓልቲ ፈትሹ ( እቲ ሓበሬታ ኣገባብ ዘለዎን ወቕታዊ ከም ዝኾነ ንምርኣይ )
💡6, ናይ ዋዛ ከይኸውን ሕተቱ ( እቲ ሓበሬታ ንዋዛ ተሓሲቡ ከይኸውን ንምርግጋጽ )
💡7, ናትኩም ኣድልዎ ወስኹሉ ( ኣብ ፍርድኹም ተጽዕኖ እንዳ ገበሩ ከምዝኾነ ንምርኣይ )
